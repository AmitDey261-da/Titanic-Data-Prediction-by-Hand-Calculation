# Titanic Data Prediction by Hand Calculation

A comprehensive PhD-level research manuscript that performs complete Titanic survival prediction — from raw data to ensemble machine learning — entirely through manual mathematical computation. No libraries, no code — every formula derived, every calculation shown step by step.

## Contents

- **Titanic_Data_Prediction_by_Hand_Calculation.md** — Full manuscript (~1,100 lines) covering:

1. **Missing Value Analysis** — percentages computed as fractions of 891
2. **Descriptive Statistics** — mean, standard deviation, quartiles via long division
3. **Skewness & Log Transformation** — z-score cubes summed manually, skew reduced from 4.79 to 0.39
4. **Correlation Analysis** — Pearson's r with full covariance computations
5. **Feature Engineering** — FamilySize, IsAlone, Title extraction
6. **Seven ML Models by Hand:**
   - Logistic Regression: sigmoid function, gradient descent iteration
   - Decision Tree: Gini impurity, information gain (Sex split IG = 0.141)
   - Random Forest: bootstrap sampling, ensemble voting
   - Gradient Boosting: sequential pseudo-residual correction
   - SVM: RBF kernel, distance computation
   - KNN: Euclidean distance, 5-neighbor voting
   - Gaussian Naive Bayes: Bayes' theorem, Gaussian likelihood
7. **Performance Metrics** — confusion matrix, precision/recall/F1, ROC-AUC
